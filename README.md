# Sankalp AI 🚀

> **Turning Intent into Action**

Sankalp AI is a multi-agent AI planning platform that transforms user goals into structured execution roadmaps. Instead of producing a single response, specialized AI agents collaborate to research, verify, strategize, and generate actionable plans.

## ✨ Features

- 🤖 Multi-Agent AI Workflow
- 🔍 Research & Knowledge Synthesis
- ✅ Risk Verification & Credibility Analysis
- 📊 Interactive SWOT Strategy Canvas
- 📅 Execution Roadmap (Kanban & Gantt)
- 📁 Drag & Drop File Upload
- 📄 TXT, PDF, DOCX, CSV & Markdown support
- 🔗 URL Analysis
- 🎯 Goal-based Planning
- 🌙 Modern Responsive UI
- 💻 Runs entirely in the browser (Simulated Mode)
